# Элементы страницы InterpretationView

## Структура элементов по порядку отображения:

### 1. Главный заголовок (keyPhrase)
**Шрифт:** Helvetica Neue Light 36pt  
**Цвет:** синий  
**Padding:** 48px горизонтально  
**Динамический элемент:** `hexagram.keyPhrase`
```
[Главный заголовок - ключевая фраза гексаграммы]
```

#### 2.1. Номер и название гексаграммы
**Шрифт:** Roboto Mono Thin 22pt  
**Цвет:** синий  
**Динамический элемент:** `hexagram.number`, `hexagram.name`
```

```

#### 2.2. Образ
**Шрифт:** Roboto Mono Thin 22pt  
**Цвет:** синий  
**Динамический элемент:** `hexagram.image` (опционально)
```

```

#### 2.3. Качества (теги)
**Шрифт:** Helvetica Neue Light 22pt  
**Цвет:** синий  
**Динамический элемент:** `hexagram.qualities` (массив строк, опционально)
```

```

#### 2.4. Разделитель (полоса)
**Высота:** 1px  
**Цвет:** синий  
**Padding:** 48px горизонтально  
**Отступ сверху:** 8px

## Примечания:

- **Все тексты выровнены по левому краю**
- **Горизонтальный padding:** 48px для большинства элементов
- **Специальный padding для буллитов:** 81px слева, 48px справа
- **Шрифты:**
  - Заголовки секций: Roboto Mono Thin 22pt
  - Основной текст: Helvetica Neue Light 22pt
  - Главный заголовок: Helvetica Neue Light 36pt
  - Кнопки ролей: Helvetica Neue Light 36pt
  - Кнопка "Продолжить": Druk Wide Cyr Medium 22pt
- **Цвет:** все элементы используют синий цвет (#286196)
- **Фон:** бежевый (#EFE7D4)
