# Книга перемен

Минималистичное iOS-приложение для работы с И-цзин (Книгой перемен).

## Технологии

- SwiftUI
- iOS 17+
- Локальное хранение (UserDefaults)
- Без backend и авторизации

## Структура проекта

```
I-CHING/
├── IChingApp.swift          # Точка входа приложения
├── ContentView.swift        # Корневой view
├── Models/
│   └── Hexagram.swift       # Модели данных (Hexagram, Line, Reading)
├── Services/
│   └── StorageService.swift # Сервис для сохранения результатов
├── Views/
│   ├── StartView.swift      # Стартовый экран
│   ├── QuestionView.swift   # Экран вопроса
│   ├── CoinsView.swift      # Экран броска монет
│   ├── HexagramView.swift   # Экран гексаграммы
│   ├── InterpretationView.swift # Экран толкования
│   └── ResultView.swift     # Экран результата
└── hexagrams.json           # Данные 64 гексаграмм
```

## Установка

1. Откройте проект в Xcode
2. Убедитесь, что `hexagrams.json` добавлен в Bundle Resources
3. Выберите симулятор или устройство iOS 17+
4. Запустите проект

## Особенности

- Минималистичный flat design
- Плавные переходы между экранами
- Локальное хранение всех результатов
- Все 64 гексаграммы с толкованиями на русском языке
