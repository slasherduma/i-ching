# Инструкция по добавлению фоновой музыки

## Формат файла
- **Формат**: `.m4a` (AAC)
- **Имя файла**: `background_music.m4a` (обязательно!)
- **Качество**: рекомендуется 128-192 kbps для баланса между качеством и размером файла

## Как конвертировать музыку в .m4a

### Вариант 1: Использовать FFmpeg (командная строка)
```bash
ffmpeg -i ваша_музыка.mp3 -codec:a aac -b:a 128k background_music.m4a
```

### Вариант 2: Использовать VLC Media Player
1. Откройте VLC
2. Медиа → Конвертировать/Сохранить
3. Добавьте ваш аудио файл
4. Нажмите "Конвертировать/Сохранить"
5. Формат: Audio - M4A (AAC)
6. Профиль: Audio AAC (128 kbps)
7. Укажите имя файла: `background_music.m4a`

### Вариант 3: Использовать онлайн конвертеры
- CloudConvert.com
- Online-Convert.com
- Выберите формат: M4A (AAC)

## Как добавить файл в Xcode проект

1. Перетащите файл `background_music.m4a` в папку `i-ching/i-ching/Audio/` в Finder
2. Откройте проект в Xcode
3. В навигаторе проекта (левая панель) найдите папку `i-ching` → `Audio`
4. Перетащите файл `background_music.m4a` из Finder в папку `Audio` в Xcode
5. **ВАЖНО**: В диалоговом окне добавления файла:
   - ✅ Отметьте "Copy items if needed"
   - ✅ Отметьте "Add to targets: i-ching"
   - Нажмите "Finish"

## Альтернативный способ (через Xcode напрямую)
1. Правой кнопкой на папку `i-ching` в навигаторе Xcode
2. Выберите "Add Files to 'i-ching'..."
3. Найдите ваш `background_music.m4a` файл
4. Убедитесь, что отмечено "Copy items if needed" и "Add to targets: i-ching"
5. Нажмите "Add"

## Проверка
После добавления файл должен:
- Отображаться в навигаторе Xcode
- Быть виден в папке `Audio` или в основном дереве проекта
- Иметь галочку в колонке "Target Membership" для target "i-ching"

## Примечания
- Музыка будет играть автоматически при запуске приложения
- Музыка будет играть в бесконечном цикле (loop)
- По умолчанию громкость установлена на 50% (0.5)
- Музыка будет продолжаться при переходе между экранами
- Музыка будет автоматически смешиваться с системными звуками благодаря категории `.ambient`
